## 1. 系统概述
易课通知识库系统是一个智能化知识管理与学习平台，提供资源管理、智能搜索、AI 辅助教学等核心功能。本手册主要介绍知识库管理和文档搜索两大模块的操作方法。

## 3. 知识库管理
通过顶部导航栏 知识库 → 资源管理 进入资源管理页面。该页面包含四个功能标签：
资源列表 — 所有资源的查看、创建、编辑、提交审核
分类管理 — 资源分类体系的维护
待审核列表 — 审核人员对资源的审核操作
删除申请列表 — 资源删除申请的审批
### 3.1 资源列表
资源列表页面是资源管理的核心，包含左侧分类树和右侧资源表格两部分。

页面布局：
资源表格字段说明：
筛选与搜索：
分类树筛选：点击左侧分类树的节点，可按分类筛选资源
搜索框：在资源列表上方的搜索框输入关键词，快速检索资源名称
分页：表格底部支持翻页，每页显示 10 条记录
### 3.2 新建资源
点击资源列表上方的 "新建资源" 按钮，弹出新建资源对话框。

表单字段说明：
操作步骤：
填写资源名称和资源类型
（可选）选择分类、填写关键词和版权信息
点击"选择文件"上传文件
（可选）填写资源描述
点击 "保存" 按钮，资源将以 草稿 状态保存
### 3.3 资源操作
在资源列表中，点击每行右侧的 "..." 操作按钮，弹出操作菜单。

操作说明：
资源也可通过列表顶部的 "提交审核" 和 "申请删除" 按钮批量操作（需先勾选资源）。
### 3.4 分类管理
点击资源管理页面上方的 "分类管理" 标签，进入分类管理页面。

功能说明：
分类树结构：
左侧显示分类的树形结构，包含父分类和子分类
点击分类节点，右侧展示分类详情（名称、父分类、子分类数量）
资源列表页面的左侧分类树与此处管理的分类体系同步
### 3.5 待审核列表
点击资源管理页面上方的 "待审核列表" 标签，进入审核管理页面。

审核流程（二级审核机制）：
资源提交 → 院校审核（初审） → 联盟审核（终审） → 审核通过
                ↓                    ↓
             驳回（反馈意见）      驳回（反馈意见）
操作步骤：
在待审核列表中查看待审资源
查看资源详情和文件内容
选择 "通过" 或 "驳回"
如驳回，需填写具体修改意见反馈给提交者
### 3.6 删除申请列表
点击资源管理页面上方的 "删除申请列表" 标签，进入删除审批页面。

注意： 资源删除遵循严格审批流程。审核通过后执行逻辑删除（资源被隐藏但数据保留），物理删除需联盟审核委员会审批，删除后不可恢复，同时系统自动记录删除日志。
### 3.7 收藏夹
在资源列表左侧边栏中点击 "收藏夹"，可查看当前用户收藏的资源列表。在资源列表中可将资源添加至收藏夹以便快速访问。

## 4. 文档搜索
系统提供两种搜索模式：关键词搜索和混合搜索（关键词 + 语义搜索），支持多条件筛选。
### 4.1 管理端搜索
通过顶部导航栏 搜索 进入文档搜索页面。

搜索功能说明：
搜索操作步骤：
在搜索框中输入关键词或自然语言描述（如"帮我找养老护理实操类视频"）
选择搜索类型（关键词搜索 / 混合搜索）
（可选）设置时间范围条件
点击 "搜索" 按钮或按回车键执行搜索
查看搜索结果，结果按相关度排序

### 4.2 学生端搜索
通过学生门户顶部导航 搜索 进入学生端搜索页面，功能与管理端搜索一致。

## 7. 资源状态流转
资源从创建到发布遵循以下完整生命周期：
┌─────────┐     提交审核     ┌─────────┐     院校通过     ┌────────────┐     联盟通过     ┌──────────────┐
│  草稿    │ ─────────────→ │  待审核  │ ─────────────→ │ 院校审核通过 │ ─────────────→ │ 联盟审核通过  │
│ (Draft)  │                │(Pending) │                │(SchoolOK)   │                │ (LeagueOK)    │
└─────────┘                └─────────┘                └────────────┘                └──────────────┘
                                │                          │                              │
                                │ 驳回                     │ 驳回                         │ 隐藏
                                ↓                          ↓                              ↓
                           ┌─────────┐                ┌─────────┐                    ┌─────────┐
                           │  已拒绝  │                │  已拒绝  │                    │  已隐藏  │
                           │(Rejected)│                │(Rejected)│                    │ (Hidden) │
                           └─────────┘                └─────────┘                    └─────────┘
                                │                                                         │
                                └──── 可重新编辑后再次提交 ──────────────────────────────→ 可恢复
版本管理说明：
对资源的修改/更新会产生新版本（v1 → v2 → v3...）
系统保留全部历史版本记录
资源更新后须重新提交审核，审核通过后替换当前版本并标注更新时间与更新内容
