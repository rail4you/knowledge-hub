### M-01 资讯分类“编码”字段含义不明确

- 来源模块：资源库 > 资讯管理 > 资讯分类管理
- 原始反馈：`编码是什么意思`
- 问题说明：新建分类弹窗里存在“编码”字段，但界面没有解释它的业务含义、填写规则、是否必填，以及它与分类名称、排序、前台展示之间的关系。对业务人员来说，这更像内部技术字段，容易造成不知道该填什么、填错是否会影响系统的困惑。
- 影响：增加分类时存在理解成本，容易产生错误录入，也会让非技术用户误以为系统要求输入"程序编码"或其他内部标识。
- **修复方案**：保存分类时自动根据名称生成唯一编码（名称转小写 + 时间戳），前端弹窗不再显示"编码"字段，用户无需理解该技术字段含义。（提交: `06ed0b3`）
