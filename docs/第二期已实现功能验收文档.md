# 第二期已实现功能验收文档

## 1. 文档说明

本文档用于对第二期当前已实现功能进行验收。验收对象为当前代码中已经完成并可交付的模块能力，不包含已明确延后的直播、RTC 面试等功能。

## 2. 验收范围

本次验收覆盖以下模块：

- 双高专业群评估支撑
- 微专业
- 就业
- 资讯
- 实训

不纳入本次验收：

- 实时视频直播
- 实时音视频面试
- 屏幕共享
- 会议录制

## 3. 验收环境建议

- 前端已部署并可访问
- API 服务正常
- 数据库迁移已执行完成
- 准备以下角色账号：
  - 联盟管理员
  - 院校管理员
  - 教师
  - 学生
  - 企业用户

## 4. 通用验收原则

每个验收点至少核验以下内容：

- 页面可正常进入
- 权限控制符合角色预期
- 数据可正确保存与回显
- 列表与详情数据一致
- 状态流转正确
- 导出文件可下载、可打开
