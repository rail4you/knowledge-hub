## 1. 概述

本开发计划针对 KnowledgeHub 平台的课程模块和学生门户进行开发，包含两个子计划：

| 文档编号 | 文档名称 | 说明 |
|----------|----------|------|
| [DEVELOPMENT_PLAN_001_01](./DEVELOPMENT_PLAN_001_01_COURSE_MODULE.md) | 课程模块开发计划 | 教师端课程创建、学生选课功能 |
| [DEVELOPMENT_PLAN_001_02](./DEVELOPMENT_PLAN_001_02_STUDENT_PORTAL.md) | 学生端页面开发计划 | 学生门户、知识库、智能问答、课程查看 |

---

## 2. 需求对照表

| 需求编号 | 需求描述 | 对应模块 | 优先级 | 状态 |
|----------|----------|----------|--------|------|
| 1 | 课程学习与进度管理：学生自主选课、断点续学、学习进度可视化 | 课程模块 | P0 | 已完成 |
| 2 | 可视化知识图谱：知识点关联查询、个性化学习路径推荐 | 图谱模块 | P1 | 已完成 |
| 3 | 智能问答模块：AI辅助问答 | [AI模块](../angular/src/app/ai/) | P0 | 已完成 |
| 4 | 章节测试与反馈：多题型测试、自动批改、测试报告 | 测试模块 | P2 | 进行中 |
| 5 | 学习数据统计：各级别数据统计与可视化 | 统计模块 | P2 | 待开始 |

## 4. 技术架构

### 4.1 技术栈

| 层级 | 技术 |
|------|------|
| 后端框架 | ASP.NET Core, ABP Framework 10 |
| 前端框架 | Angular 21, ng-zorro-antd 21 |
| 数据库 | PostgreSQL |
| 搜索引擎 | Meilisearch |
| AI服务 | Qwen Plus API |

### 4.2 模块划分

```
┌─────────────────────────────────────────────────────────────┐
│                    Angular Frontend                         │
├─────────────────────────────────────────────────────────────┤
│  Admin/Teacher UI          │      Student Portal            │

│  - 课程创建                 │      - 智能问答                │

                              │
                              ▼
```
