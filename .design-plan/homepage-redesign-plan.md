## 一、参考站点分析（ICVE 智慧职教）

### 1.1 整体设计风格

ICVE 采用**教育平台经典布局**，核心设计理念为：
- **信息密度高但不拥挤**：通过分区、卡片、标签有效组织大量内容
- **蓝色主色调**：专业、可信赖的教育行业色彩
- **模块化分区**：每个内容板块独立成区，有清晰的标题和操作入口
- **筛选标签系统**：每个板块都有分类筛选标签（pill 风格），交互直观

### 1.2 色彩体系

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 Primary | `#0052D9` / `#0056D2` | 品牌蓝，用于按钮、链接、标签激活态 |
| 主色 Hover | `#004099` | 深蓝悬停态 |
| 主色浅底 | `rgba(0,86,210,0.1)` | 标签、徽章浅色背景 |
| 背景 Base | `#f5f7fa` | 页面整体背景 |
| 卡片背景 | `#ffffff` | 白色卡片 |
| 卡片边框 | `#e5e7eb` / `#e8e8e8` | 浅灰边框 |
| 文字主色 | `#1a1a1a` | 标题、正文 |
| 文字次级 | `#666666` | 描述文字 |
| 文字辅助 | `#999999` | 辅助信息 |
| 徽章-国家级 | `#e74c3c` / 红 | 国家级项目标签 |
| 徽章-省级 | `#0052D9` / 蓝 | 省级项目标签 |
| 徽章-校级 | `#27ae60` / 绿 | 校级项目标签 |

### 1.4 关键设计组件

#### A. 区段标题栏 (Section Header)
```
┌──────────────────────────────────────────────┐
│  ◆ 资源库          [全部] [国家级] [省级] [校级] │  ← 标题左对齐，标签右对齐
└──────────────────────────────────────────────┘
```
- 标题带左侧竖线装饰（`border-left: 3px solid #0056D2`）
- 标签使用 pill 形状（`border-radius: 20px`）
- 激活标签：蓝色填充，白色文字
- 未激活：白色背景，灰色边框

#### B. 资源卡片 (Resource Card)
```
┌──────────────────────────┐
│ [图标]  资源名称          │
│         国家级 标签        │
│                          │
│ 资源描述文字...           │
│                          │
│ 📚 12 课程  📄 89 资源   │
│ ─────────────────────── │
│ 预览资源库           →   │
└──────────────────────────┘
```
- 白色背景，圆角 12px
- Hover: 上移 4px + 阴影 + 边框变蓝
- 统计信息区有上下分割线
- 底部操作区蓝色文字

#### C. 课程卡片 (Course Card)
```
┌──────────────────┐
│                  │
│   课程缩略图      │  ← 140px 高，渐变或图片
│                  │
├──────────────────┤
│ 课程名称          │
│ 学校名称          │
│ 👤 教师  👥 N人   │
└──────────────────┘
```
- 图片在上，信息在下
- 学习人数显示社交证明

#### D. 排行列表 (Ranking List)
```
┌─────────────────────────────────────────┐
│ 1  [图标] 资源库名称                      │
│    学校1, 学校2, 学校3                   │
│ 2  [图标] 资源库名称                      │
│    学校名称                              │
│ ...                                     │
│ 10 [图标] 资源库名称                      │
└─────────────────────────────────────────┘
```
- 编号 1-3 使用金/银/铜色
- 交错背景色区分行

## 二、易课通首页重设计方案

### 2.1 设计目标

1. **参考 ICVE 布局**：采用 ICVE 的分区式、卡片化布局风格
2. **适配易课通数据**：租户资源库、课程、资源、搜索等核心功能
3. **现代化视觉**：保持专业教育平台气质，同时更加现代
4. **响应式设计**：桌面端优先，兼顾平板和手机
5. **可扩展性**：后续资源库相关页面沿用此设计语言

### 2.6 技术实现方案

#### 文件变更清单

| 文件 | 操作 | 说明 |
|------|------|------|
| `angular/src/app/home/portal-home.component.ts` | 重写 | 添加轮播、标签切换、数据加载逻辑 |
| `angular/src/app/home/home.component.ts` | 保留 | 旧组件（不再使用，可后续清理） |
| `angular/src/app/home/home.component.html` | 保留 | 同上 |
| `angular/src/app/home/home.component.scss` | 保留 | 同上 |

PortalHomeComponent 目前是内联模板和样式。建议拆分为独立文件：
- `portal-home.component.ts` - 组件逻辑
- `portal-home.component.html` - 模板
- `portal-home.component.scss` - 样式

#### 数据来源

1. **资源库列表**: `PortalService.getPublicTenantList()` - 已有
2. **热门课程**: 需要后端新增 API 或使用现有接口
3. **最新资源**: 需要后端新增 API
4. **统计数据**: 可前端静态展示或后端 API
5. **Hero 轮播**: 前端静态配置 3 张轮播内容
6. **合作院校**: 前端静态配置 Logo 列表

#### Angular 实现要点

- 使用 `signal()` 管理状态
- 使用 `@if` / `@for` 控制流
- `ChangeDetectionStrategy.OnPush`
- 轮播使用纯 CSS animation 或简单的 `setInterval`
- 不使用第三方轮播库，保持轻量
- 标签切换使用 `click` 事件 + signal 状态

### 2.7 设计语言规范（后续资源库页面统一遵循）

1. **区块标题**: 24px, weight 600, 左侧 3px 蓝色竖线 (`border-left`)
2. **筛选标签**: pill 形状, 20px 圆角, 激活蓝底白字, 未激活白底灰边框
3. **卡片**: 12px 圆角, 白色背景, 1px 灰色边框, hover 上移 4px + 蓝色边框
4. **主按钮**: 蓝色填充 (#0056D2), 白色文字, 6px 圆角, 12px 24px padding
5. **次按钮/链接**: 蓝色文字, 无背景, hover 加深
6. **分割线**: 1px solid #f0f0f0
7. **间距**: section 之间 48px, card 之间 20px
8. **"更多"链接**: 蓝色文字 + → 箭头, 置于区段标题右侧
